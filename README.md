## 💻 How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/Piyush64-bit/Java-Programs.git
   cd Java-Programs
2. Open any .java file in your favorite IDE (VS Code / IntelliJ / Eclipse).

3. Compile & Run in terminal:

javac FileName.java
java FileName
